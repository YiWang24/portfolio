# MyNote - Full-Stack Note Taking App

## Overview
MyNote is a modern note-taking application built with Next.js and Node.js. It features real-time sync, markdown support, and AI-powered search.

## Tech Stack
- **Frontend**: Next.js 14, React, TailwindCSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **AI**: OpenAI API for smart search

## Key Features
- Create, edit, delete notes with rich markdown editor
- Tag-based organization
- Full-text search with AI semantic matching
- Responsive design for mobile and desktop
- Dark mode support

## What I Learned
- Server-side rendering with Next.js App Router
- MongoDB aggregation pipelines for complex queries
- Integrating OpenAI embeddings for semantic search
- Building responsive UIs with TailwindCSS
