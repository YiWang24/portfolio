# Portfolio Backend

## 项目简介

这是我的个人作品集后端，基于 Google Agent Development Kit (ADK) 构建的多 Agent 系统。

## 技术栈

- Java 21
- Spring Boot 3.2
- Google ADK 0.5.0
- WebFlux 响应式编程

## 架构亮点

### 星型拓扑
Router Agent 作为中心节点，将请求分发给专家 Agent。

### 工具系统
- GitHubTools: 展示代码能力
- RAGTools: 知识库检索
- KnowledgeTools: 语义搜索 + 联网

## 学到的东西

1. Agent 设计模式
2. Prompt Engineering
3. 向量搜索原理
4. 文件系统监控
